## 4b - Use `terraform state` to view Terraform state

To list resources within a Terraform state.

```console
terraform state list
```
```terraform 
aws_instance.foo
aws_instance.bar[0]
aws_instance.bar[1]
module.elb.aws_elb.main
```

Filtering by Resource  

```console
terraform state list aws_instance.bar
```
```terraform 
aws_instance.bar[0]
aws_instance.bar[1]
```

Filtering by Module  

```console
terraform state list module.elb
```
```terraform 
module.elb.aws_elb.main
module.elb.module.secgroups.aws_security_group.sg
```

### Terraform State Outputs  

Extract the value of an output variable from the state file.

`outputs.tf` file:
```hcl
output "instance_ips" {
  value = aws_instance.web.*.public_ip
}

output "lb_address" {
  value = aws_alb.web.public_dns
}

output "password" {
  sensitive = true
  value     = var.secret_password
}
```

```console
terraform output
```
```terraform
instance_ips = [
  "54.43.114.12",
  "52.122.13.4",
  "52.4.116.53"
]
lb_address = "my-app-alb-1657023003.us-east-1.elb.amazonaws.com"
password = <sensitive>


```console
terraform output instance_ips
```
```terraform
instance_ips = [
  "54.43.114.12",
  "52.122.13.4",
  "52.4.116.53"
]
```

```console
terraform output lb_address
```
```terraform
"my-app-alb-1657023003.us-east-1.elb.amazonaws.com"
```
